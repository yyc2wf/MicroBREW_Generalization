# MicroBREW_Generalization
## How to Run the Pipeline

This project has two steps:

1. **MATLAB** – extract frame-wise signals from the movies  
2. **R** – smooth, normalize, and find peaks  

---

### 1. MATLAB: Extract Signals



**Run**

% S2 movie
signal_table_s2 = MicroBREWS_combined('s2');

% S3 movie
signal_table_s3 = MicroBREWS_combined('s3');



---

### 2. R: Analyze Signals

**Requirements**

- The CSV files generated by MATLAB:  
  - `Movie_S2_results/signals_Movie_S2.csv`  
  - `Movie_S3_results/signals_Movie_S3.csv` 

# Analyze S2
res_s2 <- analyze_movie_signals("s2")
res_s2$peaks   # peaks for S2

# Analyze S3
res_s3 <- analyze_movie_signals("s3")
res_s3$peaks   # peaks for S3
